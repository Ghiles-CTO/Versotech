# Introducer Guide: Submitting Invoices

Learn how to submit invoices and receive commission payments.

---

## Overview

When your commissions are ready for payment, you'll be asked to submit an invoice. This guide covers the invoice requirements and submission process.

---

## When to Submit

### Invoice Request Notification

You'll be notified when:
1. Commission status is `accrued`
2. Arranger requests invoice
3. Status changes to `invoice_requested`

### How You're Notified

- Email notification
- Platform notification
- Commission status update

---

## Invoice Requirements

### Required Information

Your invoice must include:

| Element | Description |
|---------|-------------|
| **Your Details** | Name/company, address, contact |
| **Invoice Number** | Your reference number |
| **Date** | Invoice date |
| **Recipient** | Arranger/company details |
| **Description** | Commission for [deal], [investor] |
| **Amount** | Exact commission amount |
| **Bank Details** | For payment |

### Tax Information

Depending on jurisdiction:
- VAT/GST number (if registered)
- Tax amount (if applicable)
- Withholding considerations

### Currency

Invoice in the commission currency:
- As specified in fee plan
- Match exactly

## Submitting Your Invoice

### Step 1: Access the Commission

1. Go to **Commissions**
2. Find commissions with `invoice_requested` status
3. Click on the commission

### Step 2: Upload Invoice

1. Click **"Submit Invoice"**
2. Upload your invoice document (PDF preferred)
3. Verify amount matches
4. Add any notes

### Step 3: Confirm Submission

1. Review uploaded document
2. Click **"Submit"**
3. Status changes to `invoice_submitted`
4. Confirmation email sent

## Invoice Rejection

### Common Rejection Reasons

| Issue | Solution |
|-------|----------|
| Wrong amount | Correct to match commission |
| Missing details | Add required information |
| Wrong entity | Invoice correct party |
| Missing tax info | Add tax details |
| Wrong format | Use proper invoice format |

### Resubmitting

1. Note rejection reason
2. Create corrected invoice
3. Submit again
4. Follow same process

---

## Payment Processing

### After Approval

Once invoice is approved:
1. Status changes to `invoiced`
2. Payment is queued
3. Processed according to terms

### Payment Timeline

Typical timeline:
- Invoice approved
- Payment within 30 days
- Bank transfer initiated
- Funds in your account

### Payment Notification

When paid:
- Email confirmation
- Status changes to `paid`
- Platform notification

---

## Bulk Invoicing

### Multiple Commissions

If you have multiple commissions from same arranger:

Option 1: **Separate Invoices**
- One invoice per commission
- Clearer tracking
- More work

Option 2: **Combined Invoice**
- Multiple commissions on one invoice
- List each line item
- Match total to sum of commissions

### Submitting Combined

1. Create invoice with all line items
2. Go to first commission
3. Submit invoice
4. Note it covers multiple commissions
5. Repeat for other commissions (reference same invoice)

## Tax Considerations

### VAT/GST

If you're registered:
- Include tax registration number
- Add applicable tax
- Format per local requirements

### Withholding Tax

Some arrangements require withholding:
- Gross amount shown
- Withholding noted
- Net payment calculated

### Tax Forms

You may need to provide:
- W-9 (US)
- W-8BEN (international)
- Local tax certificates

## Checking Payment Status

### Tracking Payment

After submission:
1. Go to **Commissions**
2. Check status progression
3. `invoiced` = approved, pending payment
4. `paid` = payment completed

### Payment Confirmation

When paid:
- Verify in your bank
- Match to invoice
- Confirm amount correct

## Common Issues

### "Invoice keeps getting rejected"

1. Read rejection notes carefully
2. Contact arranger to clarify
3. Use provided template if available
4. Ensure all details match exactly

### "Payment is taking too long"

1. Check status (is it `invoiced`?)
2. Allow 30 days from approval
3. Contact arranger if exceeds
4. Verify bank details correct

### "Amount doesn't match what I expected"

1. Review funded amount
2. Check fee plan rate
3. Verify calculation
4. Contact if discrepancy real
