# Commercial Partner Guide: Invoicing

Submit invoices for your distribution commissions.

---

## Overview

When commissions reach `invoice_requested` status, submit your invoice to receive payment.

---

## Invoice Requirements

### Required Elements

| Element | Description |
|---------|-------------|
| **Company Details** | Legal name, address, tax ID |
| **Invoice Number** | Your reference |
| **Date** | Invoice date |
| **Recipient** | Arranger/fund details |
| **Description** | Distribution commission for [deal] |
| **Amount** | Exact commission amount |
| **Bank Details** | Wire instructions |

### Tax Compliance

Include as applicable:
- VAT/GST registration
- Tax amounts
- Withholding acknowledgment

---

## Submitting Invoices

### Step 1: Find Commission

1. Go to **Commissions**
2. Filter by `invoice_requested`
3. Click to open

### Step 2: Upload Invoice

1. Click "Submit Invoice"
2. Upload PDF
3. Verify amount matches
4. Submit

### Step 3: Track Progress

- Status: `invoice_submitted`
- Arranger reviews
- Approved â†’ `invoiced`

---

## Multiple Commissions

### Combined Invoicing

If multiple commissions from same deal/arranger:
- Can combine on one invoice
- List each line item
- Total matches sum of commissions

### Submitting

1. Create consolidated invoice
2. Submit for first commission
3. Reference for additional commissions

---

## Payment Processing

### After Approval

1. Status: `invoiced`
2. Payment queued
3. Processed per terms
4. Funds transferred

### Timeline

| Stage | Typical Time |
|-------|--------------|
| Review | 1-7 days |
| Payment | 30 days |

---

## Handling Rejections

### Common Issues

| Issue | Solution |
|-------|----------|
| Amount mismatch | Correct amount |
| Missing details | Add required info |
| Wrong format | Use proper invoice |

### Resubmission

1. Note rejection reason
2. Correct invoice
3. Resubmit
