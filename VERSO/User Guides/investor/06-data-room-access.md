## The 7-Day Auto-Grant Window

### Automatic Access Period

When your NDA is fully signed, data room access is **automatically granted** for **7 calendar days**.

```
Day 1: NDA completed ‚Üí Access auto-granted
Day 7: Access expires at 11:59 PM (auto-revoked)
```

### Auto-Grant vs Manual Grant

| Type | Trigger | Duration |
|------|---------|----------|
| **Auto-Grant** | NDA fully signed | 7 days (automatic) |
| **Manual Grant** | Staff extends access | Custom duration |
| **Extension** | Staff approves request | Additional days |

> **Note**: Staff can manually grant access or extend the auto-grant period at any time. The 7-day window is the default automatic behavior.

### Why Time-Limited?

- Protects confidential information
- Encourages timely decision-making
- Maintains security best practices
- Aligns with deal timelines

## Requesting an Extension

If you need more time:

### Step 1: Request Extension

1. Go to the **Data Room** tab
2. Click **"Request Extension"** button
3. Complete the request form:
   - Reason for extension
   - Additional time needed
   - Any questions for the arranger

### Step 2: Await Approval

1. Request goes to arranger/staff for review
2. Typical response within 1-2 business days
3. You'll be notified of the decision

### Step 3: Extension Granted

If approved:
- New expiry date confirmed by email
- Countdown resets on the platform
- Document any specific conditions

### Approval Considerations

Extensions are more likely granted when:
- You have specific due diligence questions
- You're actively progressing toward subscription
- The deal timeline permits
- You explain your situation clearly

## Common Issues

### "Data Room tab is locked"

Check that:
1. NDA is fully signed (ALL signatories)
2. Any countersignature is complete
3. Try refreshing the page
4. Contact support if still locked

### "Document won't load"

1. Try a different browser
2. Clear cache and cookies
3. Check internet connection
4. Contact support with specific document name

### "Access expired before I finished"

1. Request an extension immediately
2. Explain you need more review time
3. Extensions are often granted once

### "Can't download a document I need"

Some documents are intentionally view-only:
1. Take notes while viewing
2. Ask if download permission can be granted
3. Some documents may be provided after subscription
