# Lawyer Guide: Signing Certificates

Learn how to sign certificates and legal documents electronically.

---

## Overview

A key function of your role is signing certificates for investors when deals close. This guide covers the electronic signing process.

---

## When Signing is Required

### Trigger Events

Signature requests come when:
- Deal closes
- Investor positions created
- Certificates generated

### What You Sign

| Document | Purpose |
|----------|---------|
| **Share Certificates** | Investor ownership proof |
| **Unit Certificates** | LP unit confirmation |
| **Legal Opinions** | Certifications as needed |
| **Closing Documents** | Deal finalization |

---

## Receiving Signature Requests

### Notification

When signature needed:
1. Email with signing link
2. Platform notification
3. Appears in Signing queue

### Request Contents

Notification includes:
- Vehicle name
- Document type
- Investor details (if certificate)
- Urgency/deadline

---

## The Signing Process

### Step 1: Access Document

From email:
- Click signing link
- Opens signing interface

From platform:
1. Go to **Signing**
2. Click pending document
3. Opens for review

### Step 2: Review Document

Before signing:
1. Verify document contents
2. Check investor details
3. Confirm accuracy

### Step 3: Apply Signature

1. Navigate to signature field
2. Click to sign
3. Choose method:
   - Draw signature
   - Type name
   - Upload signature image
4. Apply

### Step 4: Complete Signing

1. Review signed document
2. Click "Complete Signing"
3. Document processed
4. Confirmation sent

---

## After Signing

### What Happens

1. Document is certified as signed
2. Sent to relevant parties
3. Stored in document system
4. Status updated

### Your Records

Signed documents appear in:
- Your signing history
- Document archive
- Available for reference

## Bulk Signing

### Multiple Certificates

For deal closings with many investors:
- Multiple certificates queued
- Sign sequentially or in batch
- Progress tracked

### Batch Process

1. Go to **Signing**
2. See all pending items
3. Process one by one
4. Or use batch signing if available

## Common Issues

### "Link Expired"

Signing links have validity period:
1. Request new link from platform
2. Or access via **Signing** section
3. Contact arranger if issues

### "Document Won't Load"

1. Try different browser
2. Clear cache
3. Check internet connection
4. Contact support

### "Signature Not Applying"

1. Ensure clicking in correct field
2. Try different signature method
3. Refresh and retry

---

## Best Practices

### Prompt Response

- Sign promptly when requested
- Delays can hold up investor documents
- Set alerts for signature requests

### Document Verification

Before signing:
- Verify investor details correct
- Check amounts and dates
- Confirm vehicle information

### Secure Access

- Keep login credentials secure
- Don't share signing access
- Use secure connection
