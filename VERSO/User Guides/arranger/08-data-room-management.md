# Arranger Guide: Data Room Management

Set up and manage secure data rooms for investor due diligence.

---

## Overview

The **Data Room** is a secure document repository where investors review confidential deal materials after signing an NDA. As an arranger, you control:
- What documents are uploaded
- How they're organized
- Who has access
- Access duration

---

## Accessing Deal Data Room

1. Navigate to your deal
2. Click the **"Data Room"** tab
3. You'll see the folder structure and documents

## Uploading Documents

### Step 1: Select Destination

1. Navigate to the target folder
2. Click **"Upload"** button

### Step 2: Choose Files

Options:
- Drag and drop files
- Click to browse
- Upload multiple at once

### Step 3: Configure Document

For each uploaded file:

| Setting | Options |
|---------|---------|
| **Name** | Display name (can differ from filename) |
| **Access Level** | Standard, Restricted, Public |
| **Download** | Allow download or view-only |
| **Version** | Mark as version if updating |

### Step 4: Complete Upload

1. Review settings
2. Click **"Upload"**
3. Document appears in folder

---

## Document Access Levels

### Standard Access

Default for most documents:
- Visible to investors with NDA
- Subject to data room access window

### Restricted Access

For highly sensitive documents:
- Requires additional approval
- May have viewing limits
- No download typically

### Public Access

For pre-NDA materials:
- Visible without NDA
- Usually just teaser/summary
- Encourages interest expression

---

## Download Controls

### Allow Download

Investor can:
- Save to their device
- Print if they wish
- Share (bound by NDA)

### View Only

Investor can only:
- View in browser
- Cannot download
- Cannot print (enforced where possible)

---

## Version Control

### Uploading New Versions

When updating a document:
1. Click on existing document
2. Click **"Upload New Version"**
3. Select updated file
4. Add version notes
5. Old version archived

### Version History

View all versions:
1. Click document **"..."** menu
2. Select **"Version History"**
3. See all versions with dates
4. Can revert to previous version

---

## Managing Access

### Who Has Access

View current access:
1. Go to **Data Room** tab
2. Click **"Manage Access"**
3. See list of investors with access

### Access Details

For each investor:
- Access granted date
- Expiry date
- Documents viewed
- Time spent

### Granting Access

Access is typically automatic after NDA signing. Manual grant:
1. Click **"Grant Access"**
2. Search for investor
3. Set access period
4. Confirm

### Revoking Access

To remove access:
1. Find investor in access list
2. Click **"Revoke"**
3. Confirm revocation
4. Investor can no longer view

---

## Access Windows

### Standard 7-Day Window

Default access period:
- Starts when NDA is fully signed
- Expires after 7 days
- Can be extended

### Custom Windows

Set different periods:
1. Go to **Data Room Settings**
2. Set default access period
3. Apply to new access grants

### Extension Requests

When investors request more time:
1. Review in **Access Requests**
2. Approve or deny
3. If approved, new expiry set

See [Data Room Extensions](../ceo-staff/06-data-room-extensions.md) for approval workflows.

## Common Issues

### "Document won't upload"

Check:
1. File size (max 100MB typically)
2. File format (PDF, Excel, Word, PPT)
3. Internet connection
4. Browser compatibility

### "Investor says document is missing"

1. Verify document is in data room
2. Check their access level
3. Ensure access hasn't expired
4. May be in restricted folder

### "Need to remove sensitive document"

1. Click document **"..."** menu
2. Select **"Delete"**
3. Confirm deletion
4. Document removed from all views
