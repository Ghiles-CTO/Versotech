2. Create account

4. Submit for approval
5. Receive approval/rejection notification
6. Check-in and customize profile

### 2. Introducer Agreement Flow
1. Receive Introducer Agreement with fee model
2. View reminders to approve
3. Approve or reject agreement
4. If approved: sign agreement digitally
5. Receive confirmation of successful signature

### 3. Introduction Tracking Flow
1. View dispatched opportunities as Introducer
2. Track investor interest status (Interested/Passed/Approved/Signed/Funded)
3. View subscription pack notifications for assigned investors
4. Track funding status
5. View transaction summary for invoice generation

### 4. Commission/Fee Payment Flow
1. View fee model per opportunity
2. Generate and send invoice to VERSO
3. Receive approval/change request notification
4. Receive payment confirmation
